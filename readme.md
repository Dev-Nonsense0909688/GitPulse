## 🚀 What is GitPulse?

GitPulse is a lightweight API that transforms GitHub data into a **compact, real-time SVG badge**.

It focuses on what actually matters:
- consistency 🔥  
- activity 📈  
- real output 💻  

---

## ✨ Features

- ⭐ Stars tracking  
- 📦 Repository count  
- 📈 Commit tracking  
- 🔥 Streak system  
- 🔀 Pull Requests  
- 🏆 Dev Score + Grade  
- 🎨 Clean SVG badge  
- ⚡ Fast (cached + threaded)

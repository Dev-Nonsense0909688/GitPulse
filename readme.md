## Usage

Generate dynamic badges for any GitHub user using this API.

### Basic Format

```
https://champagne-instrumental-mechanism-generators.trycloudflare.com/badge/<type>/<username>
```

---

### Example

```
![GitPulse](https:/champagne-instrumental-mechanism-generators.trycloudflare.com/badge/deep/Dev-Nonsense0909688)
```

---

### Available Badge Types

* `deep` → Deep Worker
* `streak` → Commit Streak
* `owl` → Night Owl
* `builder` → Project Builder
* `commit` → High activity burst

---

### Notes

* Replace `username` with your GitHub username
* Badges update automatically based on activity
* Works perfectly in GitHub READMEs
